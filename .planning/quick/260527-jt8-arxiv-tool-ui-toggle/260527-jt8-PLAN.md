# Plan: ArXiv 工具 UI 集成与开关控制

## Task 1: 修改 arxiv-tool.ts — 适配 SearchProviderConfig 模式

**Files:** `src/main/deepagent/arxiv-tool.ts`
**Action:** 修改 `createArxivTool` 函数签名，接受 `SearchProviderConfig | null` 参数（与 Tavily/AnySearch 一致）。当 config 为 null 时返回错误提示。从 config.config 中读取 `max_results` 覆盖默认值。
**Verify:** 函数签名与 createTavilyTool 一致，config 为 null 时返回友好错误
**Done:** createArxivTool 接受 SearchProviderConfig | null

## Task 2: 修改 runtime.ts — 从 tool_configs 加载 ArXiv 工具

**Files:** `src/main/deepagent/runtime.ts`
**Action:** 从 `builtInTools` 数组中移除硬编码的 `createArxivTool()`。新增 `loadToolConfig` 函数（不需要 api_key 校验），用于加载 ArXiv 等免费工具的配置。在 search tools 加载逻辑旁添加 ArXiv 工具的条件加载。
**Verify:** ArXiv 工具仅在 tool_configs 中 is_enabled=1 时被加载
**Done:** runtime.ts 不再硬编码 ArXiv 工具

## Task 3: 修改 ToolSettings.tsx — 添加 ArXiv 工具卡片

**Files:** `src/renderer/src/components/Settings/ToolSettings.tsx`
**Action:** 在 `INTEGRATED_TOOLS` 数组中添加 arxiv 条目。添加 `requiresApiKey` 可选字段到 ToolMeta 接口（默认 true），arxiv 设为 false。修改 `handleToggle` 逻辑：当 `requiresApiKey !== false` 时才检查 API Key。
**Verify:** ArXiv 卡片在设置页显示，无需 API Key 即可切换开关
**Done:** UI 中 ArXiv 工具可见且可控制
