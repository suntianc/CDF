# Wave 3: 集成层 — Master Agent 工具注册 + 端到端闭环

## Objective

将工作流工具注册到 DeepAgent 运行时，使 Master Agent 可通过 Chat 触发工作流执行。实现端到端闭环：创建 → 保存 → 执行 → 监控 → 查看历史。
