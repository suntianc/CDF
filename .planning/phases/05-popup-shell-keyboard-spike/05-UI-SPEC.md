## Interaction Contract

### 1. Trigger Condition
* **Behavior:** The popup opens when the user types `/` as the **first character** in the Master Agent chat input (textarea).
* **Constraints:** 
  * It must only open if `/` is at the beginning of the input (no preceding text other than whitespace or empty state).
  * If the input contains spaces, or exceeds 32 characters, the popup should close automatically.

### 2. Keyboard Navigation
* **ArrowUp / ArrowDown:** Moves the selection up and down in the list. Roving focus must loop around (pressing `ArrowDown` at the last item returns to the first item, and vice versa). Textarea default caret navigation must be prevented (`e.preventDefault()`).
* **Enter / Tab:**
  * Completes the highlighted command text by inserting the full command name plus a trailing space (e.g. `/goal `) into the textarea.
  * Closes the popup.
  * Focus remains in the textarea (do **not** execute/send the command in Phase 5).
  * Textarea default behaviors (Tab focus-shift, Enter form-submit) must be prevented (`e.preventDefault()`).
* **Escape:** Closes the popup. Focus must remain in the textarea without caret drift.
* **Backspace:** If the user presses Backspace when the input contains only `/`, the popup closes.
* **Shift + Enter:** Inserts a standard newline in the textarea without triggering selection or closing the popup.

### 3. IME Safety
* During CJK (Chinese, Japanese, Korean) input method editor (IME) composition, `onChange` state updates must **not** toggle or trigger the command popup.
* Input keydown events for Arrow keys/Enter/Escape during composition must be bypassed (`isComposingKeyEvent(e)`).

### 4. Layout & Positioning
* **Anchor:** The popover must anchor to the composer `<form>` wrapping the textarea using Radix's `<PopoverAnchor asChild>`.
* **Focus Override:** The popover content must prevent grabbing focus from the textarea on open (`onOpenAutoFocus={e => e.preventDefault()}`).
* **Layering:** The popover content must have `z-index >= 50` to render above the TodoList (`z-10`).
* **Filtering:** Case-insensitive substring matching on the command label. Both the query and items must be normalized using `String.prototype.normalize('NFKC')` before matching. No token splitting or fuzzy search is allowed.
