commands:readBody IPC contract（D-06）：
- 通道名：'commands:readBody'
- 入参：bodyPath: string（绝对路径）
- 返回：{ body: string; mtimeMs: number }；任何失败（路径校验失败 / 文件不存在 / I/O 异常）返回 { body: '', mtimeMs: 0 }；NEVER throw（与 commands:list 失败语义一致）

argSubstitution contract（D-02/D-03）：
- substituteArgs(body: string, ctx: { args: string; arguments?: string[] }): string
- $ARGUMENTS → ctx.args 完整字符串原样替换（CJK 不转义）
- $0 / $1 / $N → ctx.args 按 /\s+/ split 后取位置；越界保留原字面量
- $name → 仅当 name 在 ctx.arguments 中声明才替换；否则保留原字面量
- 纯 String.prototype.replaceAll + String.prototype.replace + 正则；禁止 eval / Function / 子进程
