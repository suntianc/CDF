main 端：
- function aggregateCurrentSessionContext(sessionId: string, contextLimit?: number): Promise<ContextAggregate>
  - contextLimit 缺省 → 从 db.providers (active) 读 context_limit；找不到默认 200_000
  - 每个 category 自己一个 try-catch，失败置 0 + console.warn（per Phase 7 isolation 模式）
  - projectCommandBodies：从 sessionId → projects.path → readdir `.cdf/commands/*.md` → 累加 fs.statSync(file).size × 0.25（v1.1 真实实现）
  - mcpPerTool：取 MCP collector 已加载的 tool 列表（reuse loadMcpTools），逐 tool 估算 schema 字节 × 0.25（v1.1 真实实现）
  - **Issue 1 修订**：systemPrompt / systemTools / customAgents / memoryFiles v1.1 阶段**尽力估算**（如能从 deepagent runtime 配置读，则读）；**若不能读** → 置 0 + console.warn + SUMMARY 标注「估算未实现 - default 0；v1.2 推」；**v1.1 落地不阻塞 plan**，但必须在 SUMMARY 明示这 4 类为 placeholder，不可视为 C2-01 "完整版 breakdown" 的充分证据
  - autocompactBuffer = Math.ceil(contextLimit * 0.15)
  - total = sum(所有 category 字段，**不含** freeSpace 与 autocompactBuffer)
  - freeSpace = Math.max(0, contextLimit - total - autocompactBuffer)
  - usedPct = Math.min(100, Math.round((total / contextLimit) * 100))
  - freePct = Math.round((freeSpace / contextLimit) * 100)
