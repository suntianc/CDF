## Layout

### Three-Area Structure

```
+------------------+------------------------+------------------+
|   Left Sidebar   |     Main Chat Area     |  Floating Panel  |
|   (collapsible)  |                        |  (bottom-right)  |
|   200-500px      |     flex: 1            |  340px width     |
+------------------+------------------------+------------------+
```

- **Left Sidebar**: Tree-based project/conversation navigation, collapsible to 0 (fixed handle appears when collapsed)
- **Main Chat Area**: Message history + input dialog
- **Floating Task Panel**: Fixed position bottom-right, collapsible to header-only (52px)

### Sidebar Details

- Width: 280px default, draggable 200px-500px via resize handle
- Collapse: Transitions to 0 width with fixed drawer handle (top-left) appearing
- Structure: New Chat button -> Search bar -> Action row (Plugins/Workflows) -> Tree sections (Projects, Chats) -> Bottom bar (Theme/Settings)

### Task Panel Details

- Position: fixed, bottom: 24px, right: 24px
- Size: 340px width, max-height 480px
- States: visible, collapsed (header only, 52px), hidden
- Animation: translateY transition 0.25s ease
