## Spacing Scale

All values are multiples of 4, sourced from DESIGN.md `spacing` tokens:

| Token | Value | DESIGN.md Token | Usage                             |
| ----- | ----- | --------------- | --------------------------------- |
| xs    | 4px   | `xxs`           | Icon gaps, inline padding         |
| sm    | 8px   | `xs`            | Compact element spacing           |
| md    | 16px  | `md`            | Default element spacing, card interior |
| lg    | 24px  | `lg`            | Section padding, card interior    |
| xl    | 32px  | `xl`            | Layout gaps, large card padding   |
| 2xl   | 48px  | `3xl`           | Major section breaks              |
| 3xl   | 64px  | `4xl`           | Page-level spacing                |

**Exceptions:**
- `sm` (12px, DESIGN.md `spacing.sm`) used for inline gaps between sibling elements (nav rows, button rows)
- `2xl` (40px, DESIGN.md `spacing.2xl`) available for tighter section padding if needed
- Sidebar width: 256px (between `lg` and `xl` of padding each side)

## Typography

Sourced from DESIGN.md `typography` tokens. Two font families:
- **Sans** (Geist / Inter): All display, body, button, link, label text
- **Mono** (Geist Mono / JetBrains Mono): Code blocks, technical labels, section eyebrows

| Role    | Size | Weight | Line Height | DESIGN.md Token     | Letter Spacing |
| ------- | ---- | ------ | ----------- | ------------------- | -------------- |
| Body    | 14px | 400    | 20px        | `body-sm`           | -0.28px        |
| Label   | 12px | 400    | 16px        | `caption`           | 0              |
| Heading | 16px | 600    | 24px        | `body-md-strong` вЖТ 600 | 0           |
| Display | 24px | 600    | 32px        | `display-md`        | -0.96px        |

**Additional tokens (used within components, not part of core hierarchy):**
- Body strong: 14px weight 500, line-height 20px (`body-sm-strong`) вАФ button labels, table emphasis
- Caption mono: 12px weight 400, line-height 16px (`caption-mono`) вАФ technical badges, "coming soon" labels
- Code: 13px weight 400, line-height 20px (`code`) вАФ API key display, settings code fields
- Button label: 14px weight 500, line-height 20px (`button-md`) вАФ all button labels

### Brand Gradient (Decorative, Not Functional)

The multi-stop mesh gradient (cyan вЖТ blue вЖТ violet вЖТ magenta вЖТ amber) is the brand's decorative system:
- **Reserved for:** Hero/welcome dialog atmospheric background only
- **Never:** Miniaturized to icon scale, never reduced to single color
- **Storage:** Rendered as inline SVG or CSS gradient in welcome dialog

### Accent Reserved For

The accent (`#171717` light / `#ffffff` dark) is reserved for:
1. Primary CTA buttons (`button-primary` variant)
2. Active navigation indicator (left-edge bar on sidebar row)
3. Focus ring on interactive elements
4. Selection highlight color
5. The polarity-flipped dark band (used for one-off emphasis surfaces)

**Not used for:** Secondary buttons, inactive nav items, decorative borders, loading states, badges.
