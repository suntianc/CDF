## Visual Hierarchy & Focal Points

- **Primary Focal Point:** The active delegated-task node in TaskPanel (indicated by status icon, border-strong highlight, and background-surface color).
- **Secondary Focal Point:** Pending approval actions container with a warning-themed alert box.
- **Accessibility:** Icon-only status indicators (e.g. running loader, success check, error cross) must include tooltip descriptors and `aria-label` tags for screen readers.
