## 核心决策（v3 终版）

| Topic | Decision |
|-------|----------|
| 白名单来源 | subagentIds 是唯一来源 |
| task(name) 键 | Agent.slug 唯一稳定键 |
| Agent.name | 仅 UI 展示 |

## 审查历史

### v1（初始版）
- 委派触发：混合模式
- 子Agent配置：Agent 资产库 + subagentIds
- 上下文传递：7 要素
- 结果汇报：结构化 JSON
- 执行可视化：TaskPanel

### v2（第一轮审查）
- task(tool) 格式改为 `task(name, task)`，删除 agent_id/context
- responseFormat 强制使用 Zod schema
- general-purpose 保持禁用
- 上下文扩展为 7 要素
- Runtime 包装异常状态

### v3（第二轮审查）
- subagentIds 是唯一白名单来源 ✓
- Agent.slug 是 task(name) 唯一稳定键 ✓
- Agent.name 仅用于 UI ✓
- general-purpose 始终禁用 ✓
- 无 subagentIds 时不暴露 task ✓
- 自定义 subagent 必须显式传 tools ✓
- 任务包只通过 task 参数传递 ✓
- responseFormat 统一 DelegatedTaskResultSchema ✓
- Runtime 统一包装 timeout/interrupted/parse_failed ✓
- TaskPanel 只展示 running/success/failure ✓
