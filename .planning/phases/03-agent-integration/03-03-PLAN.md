**Save button:**
```
<button
  className="btn btn-primary"
  onClick={handleSave}
  disabled={!name.trim() || saving}
>
  {saving ? <Loader2 className="w-4 h-4 animate-spin" /> : null}
  保存 Skill
</button>

**Save handler:**
```
const handleSave = async () => {
  if (!name.trim() || !scriptContent.trim()) return;
  setSaving(true);
  try {
    const id = skill?.id || crypto.randomUUID();
    await saveSkill({
      id,
      name: name.trim(),
      description: description || undefined,
      script_content: scriptContent,
      script_type: scriptType,
    });
    onSaved();
  } catch (err) {
    // error handled by store
  } finally {
    setSaving(false);
  }
};
```

**State:**

const [name, setName] = useState(skill?.name || '');

const [scriptContent, setScriptContent] = useState(skill?.script_content || '');
const [saving, setSaving] = useState(false);
```

Use `useEffect` to reset form when `skill` prop changes.

The overlay is a `pre` element positioned behind the textarea (same position, transparent textarea on top):
