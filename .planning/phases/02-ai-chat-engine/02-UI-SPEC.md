## Spacing Scale

Inherited from Phase 1 (all values are multiples of 4):

| Token | Value | Usage                                               |
| ----- | ----- | --------------------------------------------------- |
| xs    | 4px   | Icon gaps, inline padding, status tag icon gap      |
| sm    | 8px   | Message bubble inner padding, queue card inner gap  |
| md    | 16px  | Default element spacing, bubble-to-bubble gap       |
| lg    | 24px  | Section padding, input area top/bottom margin       |
| xl    | 32px  | Layout gaps, command palette padding                |
| 2xl   | 48px  | Major section breaks, welcome dialog padding        |
| 3xl   | 64px  | Page-level spacing (chat panel top padding)         |

**Exceptions for Phase 2:**
- `sm` (12px) used for inline gaps between sibling nav items in sidebar conversation list
- Message bubble max-width: 720px (between `lg` and `xl` padding each side)
- Input textarea min-height: 40px, max-height: 200px (6-8 lines at 24px line-height)
- Queue card height: auto, padding `sm` inside, gap `xs` between items
- Avatar size: 28×28px (between `xs` and `sm` each side)

### Accent Reserved For

The accent (`#171717` light / `#ffffff` dark) is reserved for:
1. Primary CTA buttons (Send button, action buttons)
2. Active navigation indicator (conversation list active item)
3. Focus ring on interactive elements in chat
4. User message bubble background
5. The polarity-flipped dark band (existing pattern from Phase 1)

**Not used for:** AI message bubbles, queue items, command palette items, tool call cards, thinking blocks, secondary buttons, decorative borders, loading states, badges.

## Visual / Interaction States

| Component      | States                                                                 |
| -------------- | ---------------------------------------------------------------------- |
| Send button    | `idle` (enabled, #171717) → `generating` (hidden, replaced by Stop) → `queued` (enabled, queued icon) |
| Stop button    | `generating` (red #ee0000, visible during AI reply) → `hidden` (idle) |
| Input textarea | `idle` (placeholder) → `typing` (has content) → `generating` (disabled, if AI replying) |
| Message bubble | `streaming` (partial content, cursor blink) → `complete` (full content) → `error` (if failed) |
| Queue item     | `pending` (in queue, ↩︎ active) → `guided` (in queue, sent, removed) → `sent` (in flow, status tag) |
| Thinking block | `thinking` (expanded, content flowing) → `complete` (auto-collapsed) |
| Tool call      | `running` (spinner) → `completed` (green check) → `error` (red, result) |
| GSD card       | `running` (spinner) → `success` (green border) → `error` (red border, retry) |
| Conversation item | `active` (highlighted) → `inactive` (normal) → `loading` (skeleton) |
