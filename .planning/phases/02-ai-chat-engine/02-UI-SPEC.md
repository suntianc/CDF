## Design System

| Property | Value |
|----------|-------|

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| `space-1` | 4px | Icon gaps, inline micro-padding |
| `space-2` | 8px | Compact element gaps, button internal padding |
| `space-3` | 12px | Sidebar top padding, dialog box internal gap |
| `space-4` | 16px | Default element spacing, message gap |
| `space-5` | 20px | Reserved |
| `space-6` | 24px | Section padding (messages horizontal, topbar vertical) |
| `space-8` | 32px | Layout gaps, dialog-footer gap |
| `space-10` | 40px | Reserved |
| `space-12` | 48px | Major section breaks |
| `space-16` | 64px | Page-level spacing |

**Sidebar width:** 280px default, resizable 200–500px range.
**Touch target minimum:** 32x32px (dialog buttons), 44px recommended for icon-only targets.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Display | 28px | 700 (bold) | 1.3 |
| Heading | 16px | 600 (semibold) | 1.3 |
| Body/Label | 14px | 400–500 (regular–medium) | 1.7 |
| Small/Meta | 11px | 500 (medium) | 1.4 |

> **Consolidation note:** Label (12px) merged into Body (14px). Code (13px) merged into Body (14px). Final 4-size scale enforced per Registry Safety rule.

**Font stacks:**
- Sans: `'Inter', -apple-system, BlinkMacSystemFont, system-ui, sans-serif`
- Mono: `'JetBrains Mono', 'SF Mono', 'Fira Code', monospace`

**Source:** `dashboard.html` line 48 (`font-size: 14px; line-height: 1.5`), `codex-onboarding.html` display headline 28px bold.
